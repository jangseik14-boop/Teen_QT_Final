
# 🚀 예본TeenQT 배포 가이드 (보안 패치 완료)

네트리파이의 보안 취약점 블로킹 문제를 해결하기 위해 Next.js 버전을 15.4.1로 업데이트했습니다.

### ✅ 최종 배포 절차 (매우 중요)
1. 좌측의 **[↑] 버튼**을 눌러 코드를 **Sync** 하세요. (새로운 버전이 깃허브로 전달됩니다.)
2. 네트리파이(Netlify) 화면에서 **[Deploys]** 탭을 클릭합니다.
3. 실패한 빌드 목록 근처의 **[Trigger deploy]** 버튼을 누릅니다.
4. 나타나는 메뉴에서 반드시 **[Deploy project without cache]**를 클릭하세요! (사진 속 바로 그 메뉴!)

이 버튼을 누르면 과거의 보안 오류 기록을 완전히 지우고 깨끗하게 새 버전으로 설치가 진행됩니다.

### ✅ 환경 변수 확인
네트리파이 설정에서 아래 변수가 잘 들어있는지 마지막으로 확인해주세요:
- `GEMINI_API_KEY`: (사용자님의 키)
- `NEXT_PUBLIC_FIREBASE_PROJECT_ID`: `studio-5290180250-baff5`
- `NODE_VERSION`: `20`
