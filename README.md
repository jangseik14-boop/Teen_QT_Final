
# 🎉 예본TeenQT 배포 긴급 해결 가이드 (버튼 안 눌릴 때)

빌드 에러(`exit status 1`)가 발생했거나 [만들기] 버튼이 안 눌린다면 아래 방법을 따라주세요.

### 🚀 [만들기] 버튼 활성화 팁
1. **브랜치 입력창**에 적힌 글자를 지우고 다시 **`main`**을 입력합니다.
2. 입력창 아래에 뜨는 **`main` 목록을 마우스로 직접 클릭**하세요. (직접 타이핑만 하면 인식이 안 될 수 있습니다.)
3. 빨간색 테두리가 사라지면 **[만들기]** 버튼이 파란색으로 변하며 클릭 가능해집니다.

### 🛠️ 해결이 안 된다면?
- 창을 닫고 **새로고침(F5)**을 한 번 한 뒤 다시 시도해 보세요.
- 제가 메모리를 2048MiB로 넉넉하게 늘려두었으니, 이번에 빌드가 시작되면 무사히 성공할 것입니다.
