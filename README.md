
# 🚀 예본TeenQT 배포 및 관리 가이드

앱 배포 중 발생한 문제를 해결하고 AI 기능을 활성화하는 방법입니다.

## 1. 환경 변수(API 키) 등록 위치 (중요!)
현재 화면에서 환경 변수 칸이 안 보인다면 다음 순서로 진행하세요:

1.  **새로고침**: 브라우저 페이지를 **새로고침(F5)** 합니다.
2.  **설정 진입**: 상단 메뉴의 **[설정 (Settings)]** -> 왼쪽 메뉴의 **[환경 (Environment)]**을 차례로 누릅니다.
3.  **환경 선택**: 목록에 나타난 **`prod`** 라는 이름을 클릭합니다. (입력창이 아니라 목록에 있는 글자를 눌러야 합니다.)
4.  **변수 추가**: 드디어 나타나는 **환경 변수(Environment variables)** 섹션에서 **[Add variable]**을 클릭합니다.
5.  **정보 입력**:
    *   **이름(Key)**: `GEMINI_API_KEY`
    *   **값(Value)**: 복사해둔 API 키 붙여넣기
6.  **저장**: 하단의 **[저장]** 버튼을 눌러 완료합니다.

## 2. 다시 배포하기 (빌드 오류 해결)
환경 변수를 등록한 후에는, 화면 우측 상단의 **[출시 만들기]** 버튼을 눌러 다시 배포를 시도하세요. API 키가 등록되었기 때문에 이번에는 성공할 것입니다.
