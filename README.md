
# 🎉 예본TeenQT 최종 배포 가이드 (성공까지 1분 전!)

데이터베이스 연동이 성공한 것을 확인했습니다! 이제 마지막 빌드만 완료하면 됩니다.

### 🚀 마지막 빌드 성공을 위한 조작법
1.  **App Hosting** 메뉴로 이동합니다.
2.  **[출시 만들기]** 버튼을 클릭합니다.
3.  **브랜치 선택 (매우 중요)**:
    *   입력창에 `main`을 직접 치면 빨간 테두리가 유지될 수 있습니다.
    *   `m`만 쳐도 아래에 **`main`**이라는 선택지가 뜹니다.
    *   **마우스로 그 `main` 글자를 반드시 클릭**하세요.
4.  빨간 테두리가 사라지고 **[만들기]** 버튼이 활성화되면 클릭합니다.

### ⏳ 빌드 중 (약 5분 소요)
- 빌드가 진행되는 동안은 페이지를 새로고침하지 말고 기다려 주세요.
- `stand-alone` 모드와 넉넉한 메모리(2GB) 설정으로 이번에는 무사히 통과할 것입니다.

### ✅ 완료 후
- 초록색 체크 표시가 뜨면 상단의 **도메인 주소**를 클릭하여 앱에 접속하세요!
