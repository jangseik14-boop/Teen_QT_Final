
# 🚀 예본TeenQT 배포 및 관리 가이드

앱 배포 중 발생한 문제를 해결하고 AI 기능을 활성화하는 최종 단계입니다.

## ⚠️ [저장] 버튼이 안 눌리거나 메뉴가 안 보일 때 (필독!)
1.  **글자 다시 쓰기**: `prod`를 지우고 `production`으로 입력해 보세요.
2.  **새로고침**: `F5`를 누른 뒤 왼쪽 메뉴의 **[App Hosting] -> [설정] -> [환경]**으로 다시 들어오세요.
3.  **목록 클릭**: 이미 저장되었다면 아래쪽에 `prod`라는 목록이 보일 겁니다. 그 **글자 자체를 클릭**해야 상세 메뉴(환경 변수)가 나옵니다.

## 1. 환경 변수(API 키) 등록 위치 (상세)
현재 **[설정] -> [환경]** 화면에 계시다면 다음을 확인하세요:

1.  **환경 이름**: `prod` 입력 후 저장 (이미 했다면 패스)
2.  **목록 선택**: 화면 중앙에 나타난 `prod` 환경 리스트를 클릭합니다.
3.  **변수 추가**: 아래로 펼쳐지는 메뉴에서 **환경 변수(Environment variables)** 섹션의 **[Add variable]**을 누릅니다.
    *   **이름(Key)**: `GEMINI_API_KEY`
    *   **값(Value)**: 본인의 API 키 붙여넣기
4.  **저장**: 하단의 **[저장]** 버튼을 눌러 완료합니다.

## 2. 다시 배포하기 (빌드 오류 해결)
환경 변수를 등록한 후에는 반드시 다음을 수행해야 합니다:
1.  상단 메뉴의 **[출시]** 탭을 클릭합니다.
2.  우측 상단의 **[출시 만들기 (Create release)]** 버튼을 누릅니다.
3.  API 키가 등록된 상태로 다시 빌드가 시작되며, 이번에는 성공할 것입니다.
