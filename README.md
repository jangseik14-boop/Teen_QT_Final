
# 🚀 예본TeenQT 배포 및 관리 긴급 가이드

앱 설정 중 화면이 멈추거나 버튼이 안 눌릴 때 아래 순서대로 진행해 보세요!

## 📍 [핵심] 로딩 바가 돌아가고 있을 때!
*   **기다리지 마세요!**: 지금 로딩 바가 돌아가고 있어도 상단의 **[설정(Settings)]** 탭은 누를 수 있습니다.
*   **미리 등록**: 빌드가 끝나기 전에 미리 `GEMINI_API_KEY`를 등록해두면 다음 빌드 때 바로 적용됩니다.

---

## 🚨 [환경 변수] 칸이 안 보일 때 (가장 중요!)
1.  **목록을 클릭하세요**: 환경 이름을 만들고 저장했다면, 화면 아래쪽에 생성된 `prod` 또는 `production`이라는 **글자 자체를 클릭**해야 합니다. 
2.  **펼쳐지는 메뉴**: 글자를 클릭하면 그제서야 아래로 **[환경 변수(Environment variables)]** 섹션과 **[Add variable]** 버튼이 나타납니다.
3.  **새로고침(F5)**: 아무리 눌러도 반응이 없다면 새로고침을 한 번 하고 다시 시도하세요.

---

## 1. 환경 변수(API 키) 등록 최종 단계
1.  상단 메뉴의 **[설정(Settings)]** -> 왼쪽 **[환경(Environment)]** 클릭
2.  목록에 있는 `production` (또는 본인이 만든 이름) **글자 클릭**
3.  펼쳐진 메뉴에서 **[Add variable]** 클릭
    *   **Key**: `GEMINI_API_KEY`
    *   **Value**: 본인의 AI API 키 붙여넣기
4.  하단의 **[저장]** 클릭

## 2. 빌드 오류 해결하기
환경 변수를 등록한 후에는 반드시 다시 빌드해야 합니다.
1.  상단의 **[개요(Overview)]** 또는 **[출시(Release)]** 탭 클릭
2.  우측 상단의 **[출시 만들기(Create release)]** 클릭
3.  이제 API 키가 인식되어 정상적으로 배포가 완료됩니다!
