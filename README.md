
# 🚀 예본TeenQT 배포 및 관리 가이드

앱 배포 중 발생한 문제를 해결하고 AI 기능을 활성화하는 최종 단계입니다.

## ⚠️ [저장] 버튼이 안 눌리거나 메뉴가 안 보일 때 (필독!)
1.  **[취소] 버튼 클릭**: 현재 화면에서 **[취소]**를 먼저 눌러보세요. 이름 입력 모드에서 빠져나와야 아래쪽 메뉴가 보입니다.
2.  **글자 다시 쓰기**: `production`을 지우고 `prod`라고 입력해 보세요. 파란색 [저장] 버튼이 활성화되면 클릭합니다.
3.  **목록 클릭**: 저장을 누른 후 아래쪽에 `prod`라는 목록이 보이면, 그 **글자 자체를 클릭**해야 상세 메뉴(환경 변수)가 나옵니다.

## 1. 환경 변수(API 키) 등록 위치 (상세)
현재 **[설정] -> [환경]** 화면에 계시다면 다음을 확인하세요:

1.  **목록 선택**: 화면 중앙에 나타난 `production` 또는 `prod` 환경 리스트를 클릭합니다.
2.  **변수 추가**: 아래로 펼쳐지는 메뉴에서 **환경 변수(Environment variables)** 섹션의 **[Add variable]**을 누릅니다.
    *   **이름(Key)**: `GEMINI_API_KEY`
    *   **값(Value)**: 본인의 API 키 붙여넣기
3.  **저장**: 하단의 **[저장]** 버튼을 눌러 완료합니다.

## 2. 다시 배포하기 (빌드 오류 해결)
환경 변수를 등록한 후에는 반드시 다음을 수행해야 합니다:
1.  상단 메뉴의 **[출시]** 탭을 클릭합니다.
2.  우측 상단의 **[출시 만들기 (Create release)]** 버튼을 누릅니다.
3.  API 키가 등록된 상태로 다시 빌드가 시작되며, 이번에는 성공할 것입니다.
