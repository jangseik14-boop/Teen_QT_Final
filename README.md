1. **백엔드 ID**: 입력창에 **`teenqt`** (소문자)를 입력하고 **[다음]**을 누르세요.
2. **웹 앱 연결**: 목록에서 현재 프로젝트를 선택하고 **[완료]** 버튼을 누르세요.
3. **빌드 시작**: 이제 화면에 로딩 표시가 뜹니다. 
   - **중요**: 초록색 체크(`성공`)가 뜰 때까지 **약 5분**간 기다려 주세요.
   - 성공하면 화면 상단에 생성된 `https://...` 주소를 클릭해 앱에 접속하세요!
