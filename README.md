
# 🏁 예본TeenQT 배포 완료 가이드

### 1. 지금 화면에 보이는 건 무엇인가요?
- **파란색 뱅글뱅글**: 구글 서버가 앱을 만드는 중입니다 (정상!).
- **[보기] 버튼**: 클릭하면 어떤 작업이 진행 중인지 자세히 볼 수 있습니다.

### 2. 배포가 끝나면 어떻게 하나요?
1. 초록색 체크 표시가 뜰 때까지 기다립니다.
2. 카드 맨 위에 있는 **영어 주소(URL)**를 클릭합니다.
3. 짜잔! 여러분의 **예본TeenQT**가 인터넷에 공개되었습니다.

### 3. AI가 작동하지 않는다면?
- `apphosting.yaml` 파일에 본인의 **GEMINI_API_KEY**를 넣었는지 다시 확인해 주세요.
- 키를 넣고 저장하면 자동으로 다시 배포가 시작됩니다.
