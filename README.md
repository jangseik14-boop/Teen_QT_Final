
# 🚀 예본TeenQT 배포 및 관리 가이드

앱 배포 중 발생한 문제를 해결하고 AI 기능을 활성화하는 방법입니다.

## 1. 환경 변수(API 키) 등록 위치 찾기
현재 보고 계신 **Firebase 콘솔 화면**에서 다음을 수행하세요:
1.  중앙의 탭 메뉴(`개요`, `출시`, `로그`, `사용량`, `설정`) 중 가장 오른쪽에 있는 **[설정 (Settings)]**을 클릭합니다.
2.  왼쪽 사이드바(또는 화면 중앙)에 나타나는 메뉴 중 **[환경 (Environment)]**을 선택합니다.
3.  이미 `prod` 환경을 만드셨다면, 그 아래에 **환경 변수(Environment variables)** 섹션이 보입니다.
4.  거기서 **[Add variable]**을 눌러 `GEMINI_API_KEY`를 등록하세요.

## 2. 빌드 오류 해결하기
화면에 빨간색으로 **"빌드에 오류가 발생했습니다"**라고 뜨는 이유는 보통 다음과 같습니다:
*   **API 키 누락**: 일부 페이지가 빌드 중에 AI를 호출하려고 하는데 키가 없어서 실패할 수 있습니다. 위 1번 과정을 먼저 완료해 주세요.
*   **상세 원인 확인**: 화면 우측 하단의 **[Cloud Build 로그 보기]**를 클릭하면 정확히 어떤 파일에서 오류가 났는지 영어로 나옵니다.

## 3. 다시 배포하기
환경 변수를 등록한 후에는, 화면 우측 상단의 **[출시 만들기]** 버튼을 눌러 다시 배포를 시도해 보세요.

## 4. 관리자 비밀번호
- 모든 설정이 완료된 후 앱에 접속하여 **MY** 탭 -> **관리자 모드** 진입 시 비밀번호는 `141414` 입니다.
