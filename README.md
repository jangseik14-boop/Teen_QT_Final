
# 🚀 예본TeenQT 배포 및 관리 최종 가이드

빌드 오류가 발생했을 때 당황하지 마세요! 아래 순서대로만 하면 100% 성공합니다.

## 🛠️ [중요] 빌드 오류 해결법
최근 Next.js 버전의 보안 이슈로 배포가 실패하는 경우가 있습니다. 
제가 이미 코드를 수정하여 **Next.js 버전을 15.1.7로 변경**해 두었으니, 잠시 기다리시면 자동으로 다시 빌드가 시작됩니다.

## 🔑 [최종] API 키 입력하기 (가장 확실한 방법!)
콘솔 UI에서 환경 변수 버튼을 찾기 힘들다면, 파일에서 직접 수정하는 게 가장 빠릅니다.

1.  왼쪽 파일 목록에서 **`apphosting.yaml`** 파일을 클릭하세요.
2.  가장 아래 줄에 있는 `GEMINI_API_KEY` 의 **`value: "여기에..."`** 부분을 지우고 본인의 API 키를 붙여넣으세요.
    *   예: `value: "AIzaSy..."` (반드시 따옴표 안에 넣으세요!)
3.  파일을 저장하면 다시 빌드가 시작되고, 배포 완료 후 AI 기능이 작동합니다.

## 📍 [참고] 콘솔에서 하고 싶을 때 팁
- **설정** 탭 -> **환경** 메뉴 클릭
- 이름 입력 칸 옆의 **[취소]**를 누르거나, 아래에 생긴 **`production` 글자 자체를 클릭**해야 변수 추가 버튼이 나타납니다.
