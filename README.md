
# 🚀 예본TeenQT 배포 및 관리 긴급 가이드

콘솔 화면에서 환경 변수 버튼이 안 보일 때의 **가장 확실한 해결책**입니다!

## 🛠️ [최종] 코드로 해결하기 (가장 확실!)
콘솔 UI가 먹통이라면, 우리 코드 파일(`apphosting.yaml`)에서 직접 키를 넣어줄 수 있습니다.

1.  왼쪽 파일 목록에서 **`apphosting.yaml`** 파일을 클릭하세요.
2.  가장 아래 줄에 있는 `GEMINI_API_KEY` 의 **`value: "여기에..."`** 부분을 지우고 본인의 API 키를 붙여넣으세요.
    *   예: `value: "AIzaSy..."` (따옴표 안에 넣으셔야 합니다!)
3.  파일을 저장하면 제가 자동으로 다시 배포를 준비합니다.

---

## 📍 [참고] 콘솔에서 꼭 하고 싶으시다면
1.  **새로고침(F5)**을 하세요.
2.  이름 입력 칸이 아니라, 아래에 생긴 **`production` 이라는 글자 자체를 클릭**하세요.
3.  글자를 클릭해야 그 아래로 **[변수 추가]** 메뉴가 나타납니다.
