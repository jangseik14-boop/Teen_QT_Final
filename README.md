
# 🎉 예본TeenQT 배포 긴급 해결 가이드

빌드 에러(`exit status 1`)를 해결하기 위해 메모리 최적화 설정을 완료했습니다. 아래 방법으로 다시 시도해 주세요.

### 🚀 [만들기] 버튼 활성화 및 배포 방법
1. **브랜치 입력창**에 **`main`**을 입력합니다.
2. 입력창 아래에 뜨는 **`main` 목록을 마우스로 반드시 클릭**하세요. (직접 타이핑만 하면 버튼이 활성화되지 않습니다.)
3. 빨간색 테두리가 사라지고 **[만들기]** 버튼이 파란색으로 변하면 클릭합니다.

### 🛠️ 여전히 안 된다면?
- 브라우저의 **시크릿 모드(Ctrl+Shift+N)**에서 파이어베이스 콘솔에 접속해 보세요. (캐시 문제일 수 있습니다.)
- 빌드 로그에서 `standalone` 모드가 활성화되어 더 가볍게 빌드될 것입니다.
