
# 🚀 예본TeenQT 배포 및 관리 가이드

앱 배포 중 발생한 문제를 해결하고 AI 기능을 활성화하는 최종 단계입니다.

## 1. 환경 변수(API 키) 등록 위치 찾기 (메인 화면 기준)
현재 Firebase 메인 화면이라면 다음 순서로 이동하세요:

1.  **메뉴 이동**: 왼쪽 사이드바 메뉴에서 **[App Hosting]**을 클릭합니다.
2.  **백엔드 선택**: 목록에 나타난 본인의 백엔드 이름(예: `teen-qt-backend`)을 클릭합니다.
3.  **설정 진입**: 상단 메뉴 탭 중에서 **[설정 (Settings)]**를 클릭합니다.
4.  **환경 선택**: 왼쪽 메뉴의 **[환경 (Environment)]**을 누릅니다.
5.  **변수 추가**: 목록에 있는 **`prod`**를 클릭한 후, 아래에 나타나는 **환경 변수(Environment variables)** 섹션에서 **[Add variable]**을 누릅니다.
    *   **이름(Key)**: `GEMINI_API_KEY`
    *   **값(Value)**: 본인의 API 키 붙여넣기
6.  **저장**: 하단의 **[저장]** 버튼을 눌러 완료합니다.

## 2. 다시 배포하기 (빌드 오류 해결)
환경 변수를 등록한 후에는, 해당 화면 우측 상단의 **[출시 만들기 (Create release)]** 버튼을 눌러 다시 배포를 시도하세요. API 키가 등록되었기 때문에 이번에는 성공할 것입니다.
