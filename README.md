
# 🎉 예본TeenQT 배포 완료를 위한 마지막 체크리스트

지금 화면에서 **[시작하기]**를 누른 후, 아래 순서대로 진행하면 5분 뒤에 앱이 세상에 나옵니다!

### 🚀 App Hosting 설정 가이드
1. **GitHub 연결**: 본인의 GitHub 계정을 연결하고, 이 프로젝트의 저장소를 선택하세요.
2. **브랜치 설정 (중요)**:
   - 입력창에 `main`을 입력합니다.
   - **반드시** 입력창 아래에 나타나는 **`main`** 글자를 마우스로 클릭하세요! (빨간 테두리가 사라져야 합니다.)
3. **환경 변수**: `apphosting.yaml`에 이미 설정을 해두었으니, 추가 입력 없이 **[다음]** 또는 **[만들기]**를 누르세요.
4. **빌드 대기**: 뱅글뱅글 돌아가는 로딩 표시가 초록색 체크(`성공`)로 바뀔 때까지 기다립니다 (약 5분).
